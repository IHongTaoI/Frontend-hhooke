# :watermelon: 基础

## :boom: JavaScript对象

### :mount_fuji: 什么是对象

可以使用两种形式来创建对象: 声明形式和构建形式。

```js
// 声明形式
var  obj = {
  key: 1,
}

// 构建形式
var obj2 = new Object();
obj2.key = 1;
```

### :mount_fuji: js 数据类型

- number
- boolean
- null
- undefined
- object
- symbol
- BigInt
